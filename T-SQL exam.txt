1. Create a Select statement to generate the following result  (note: in SellingPrice use Lotprice if type is Lot only else use Houseprice)

SELECT  
DISTINCT [customer].CustomerNo,
         [customer].UnitID,
         [customer].LastName,
         [customer].FirstName,
         [customer].MiddleName,
         [dtl].EquityTerm,
         [dtl].EquityAmount,
         [dtl].MATerm,
         [dtl].MAAmount,
         [dtl].MIR,
         [dtl].FIRE,
         [dtl].LoanAmt,
         [dtl].IntRate,
         IIF([inv].Type = 'Lot Only', [inv].Lotprice, [inv].HousePrice) AS [SellingPrice]

FROM    CustomerTbl [customer], CustomerDtl [dtl], InventoryTbl [inv]
WHERE   [customer].CustomerNo = [dtl].CustomerNo
and     [customer].UnitID = [inv].UnitID
